# Secure Hospital Management System

## Overview
A secure Electronic Health Record (EHR) database system designed to protect sensitive patient information using modern database security mechanisms.

## Features
- Role-Based Access Control (RBAC)
- AES Encryption for Medical Records
- Audit Logging and Accountability
- Database Replication for Disaster Recovery
- HIPAA-aligned Security Design

## Technologies Used
- MySQL
- SQL
- AES Encryption
- RBAC
- Database Replication

## Security Concepts
- Least Privilege
- Access Control
- Data Confidentiality
- Integrity Protection
- Audit Trails
- Disaster Recovery
